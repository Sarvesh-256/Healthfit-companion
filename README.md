# 🧠 HealthFit Companion

An AI-based Health & Fitness System that uses **Fuzzy Logic** and **Artificial Neural Networks (ANN)** to analyze user health and provide personalized diet and wellness recommendations.

---

## 🚀 Features

- 📊 BMI, BMR, and TDEE Calculation  
- 🧠 Mental Health Analysis using Fuzzy Logic  
- 🏋️ Fitness Level Prediction using ANN  
- 📈 Wellness Score Calculation  
- 🍽️ Personalized Diet Recommendation  
- 💾 Daily Health Data Storage  
- 📉 Progress Visualization (Graph)

---

## 🧩 Technologies Used

- Python  
- Flask (Backend)  
- HTML, CSS, JavaScript (Frontend)  
- scikit-fuzzy (Fuzzy Logic)  
- scikit-learn (ANN)  
- pandas, numpy  
- Matplotlib (Visualization)
